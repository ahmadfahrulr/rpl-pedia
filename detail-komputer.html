Materi Pengenalan Komputer

Memahami fondasi dari semua teknologi digital: perangkat keras dan perangkat lunak.

Perangkat Keras (Hardware)

Semua komponen fisik dari sebuah sistem komputer yang dapat dilihat dan disentuh. Hardware adalah "tubuh" dari komputer.

Komponen Utama:

  • CPU (Central Processing Unit): Otak komputer yang mengeksekusi semua perintah dan pemrosesan data.
  • RAM (Random Access Memory): Memori sementara berkecepatan tinggi untuk data yang sedang aktif digunakan.
  • Penyimpanan (Storage): Tempat menyimpan data secara permanen, seperti Hard Disk Drive (HDD) atau Solid State Drive (SSD).
  • Perangkat Input: Alat untuk memasukkan data ke komputer (contoh: keyboard, mouse).
  • Perangkat Output: Alat untuk menampilkan hasil dari komputer (contoh: monitor, printer).

Perangkat Lunak (Software)

Kumpulan instruksi atau program yang tidak berwujud, yang memberitahu hardware apa yang harus dilakukan. Software adalah "jiwa" dari komputer.

Jenis Utama:

  • Sistem Operasi (OS): Perangkat lunak paling penting yang mengelola semua sumber daya komputer. Contoh: Windows, macOS, Linux, Android.
  • Aplikasi: Program yang dirancang untuk melakukan tugas spesifik bagi pengguna. Contoh: Browser Web, Microsoft Office, Adobe Photoshop, Game.
