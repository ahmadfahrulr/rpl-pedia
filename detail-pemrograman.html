Materi Dasar Pemrograman

Fondasi utama untuk membangun logika, memecahkan masalah, dan menciptakan perangkat lunak.

Algoritma & Struktur Data

Jantung dari pemrograman. Algoritma adalah langkah-langkah logis untuk menyelesaikan masalah, sementara struktur data adalah cara efisien untuk menyimpan dan mengatur data.

Konsep Utama:

  • Logika Percabangan: Menggunakan `if-else` untuk membuat keputusan.
  • Perulangan: Menggunakan `for` dan `while` untuk tugas berulang.
  • Struktur Data: Memahami Array, List, Stack, Queue, dan Map.
  • Kompleksitas Waktu: Mengukur efisiensi sebuah algoritma.

Python

Bahasa yang ramah bagi pemula dengan sintaks yang bersih dan mudah dibaca. Sangat populer untuk pengembangan web, analisis data, kecerdasan buatan (AI), dan otomatisasi.

Contoh Kode Sederhana:

# Program untuk menyapa pengguna
nama = input("Masukkan nama Anda: ")
print(f"Halo, {nama}! Selamat belajar Python.")

Java

Bahasa yang kokoh, berorientasi objek, dan dapat berjalan di berbagai platform (Write Once, Run Anywhere). Banyak digunakan untuk aplikasi Android, sistem enterprise skala besar, dan backend.

Contoh Kode Sederhana:

// Program untuk menampilkan pesan
class HaloDunia {
    public static void main(String[] args) {
        System.out.println("Halo, Dunia! Belajar Java.");
    }
}
