Materi Basis Data

Cara mengelola, menyimpan, dan mengambil informasi secara terstruktur dan efisien.

SQL: Database Relasional

Structured Query Language (SQL) adalah bahasa standar untuk berinteraksi dengan database relasional, di mana data disimpan dalam tabel-tabel yang saling berhubungan. Contoh populernya adalah MySQL dan PostgreSQL.

Perintah Dasar:

  • SELECT: Mengambil data dari tabel.
  • INSERT: Menambahkan data baru ke dalam tabel.
  • UPDATE: Mengubah data yang sudah ada.
  • DELETE: Menghapus data dari tabel.

Contoh Query:

-- Mengambil nama dan email dari tabel pengguna
SELECT nama, email FROM pengguna WHERE kota = 'Jakarta';

NoSQL: Database Non-Relasional

Database NoSQL (Not Only SQL) dirancang untuk data yang tidak terstruktur dan dapat diskalakan dengan mudah. Data disimpan dalam format yang fleksibel seperti dokumen JSON. Contoh populernya adalah MongoDB.

Karakteristik Utama:

  • Skema Fleksibel: Tidak memerlukan struktur tabel yang kaku.
  • Skalabilitas Horizontal: Mudah untuk ditambah server baru.
  • Kinerja Tinggi: Cepat untuk operasi baca dan tulis data dalam jumlah besar.

Contoh Dokumen (JSON):

{
  "_id": "123",
  "username": "budi",
  "email": "budi@email.com",
  "hobi": ["membaca", "coding"]
}
