Materi Pengembangan Web

Membangun situs web dan aplikasi interaktif yang berjalan di peramban (browser).

Front-End: Tampilan Visual

Front-end adalah segala sesuatu yang dilihat dan diinteraksikan langsung oleh pengguna di browser. Tujuannya adalah menciptakan antarmuka yang menarik, responsif, dan mudah digunakan.

Teknologi Utama:

  • HTML: Kerangka atau struktur dari sebuah halaman web.
  • CSS: Menata dan mempercantik tampilan (warna, layout, font).
  • JavaScript: Memberikan interaktivitas dan logika di sisi klien.

Framework Populer:

React, Vue, dan Svelte adalah beberapa framework JavaScript yang populer untuk mempercepat pengembangan front-end.

Back-End: Logika Server

Back-end adalah "mesin" yang bekerja di balik layar. Ia bertanggung jawab mengelola data, memproses permintaan dari pengguna, dan berkomunikasi dengan database.

Teknologi Utama:

  • Bahasa Pemrograman: Node.js (JavaScript), PHP, Python, Java, Go.
  • Database: Berinteraksi dengan database SQL atau NoSQL.
  • API (Application Programming Interface): Jembatan komunikasi antara front-end dan back-end.

Framework Populer:

Express (untuk Node.js), Laravel (untuk PHP), dan Django (untuk Python) adalah beberapa framework yang mempercepat pengembangan back-end.
